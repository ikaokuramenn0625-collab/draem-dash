自己認識は高いのに行動に移せない人向け — ジャーナルを今日のタスクに変換
痛み: 毎日ジャーナル書いてアイデアは山ほど、でも実際の output はゼロに近い。頭の中の思考と手の動きのギャップが大きい。
価値: 5分のジャーナル → 3-5個の具体アクション。思考が終わった瞬間に「今日やること」に変わる。
本番版で追加される機能:
・Claude API で文脈理解（今は heuristic）
・音声入力（喋る → 起こす → アクション化）
・Notion / GTD ツールへ自動連携
・「やったかどうか」を夜に自動リマインド
Source: r/productivity post
